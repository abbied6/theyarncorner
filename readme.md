# The Yarn Corner

## Overview
The Yarn Corner is a beginner-friendly website designed to help users learn the basics of crochet and knitting. It serves as a central hub for tutorials, essential techniques, supply guidance, and a wide variety of patterns for both crafts. The goal of this project is to make fiber arts more accessible and less intimidating for new learners.

## Features
* Step-by-step guidance for beginner crochet and knitting techniques
* Dedicated pages for both crochet and knitting content
* Collection of patterns for users to practice and build skills
* Information on essential tools and supplies
* Multi-page layout with clear and simple navigation
* Responsive design for usability across devices

## Pages
* `index.html` – Home page and introduction
* `crochet.html` – Crochet basics and beginner tutorials
* `knit.html` – Knitting basics and beginner tutorials
* `patterns.html` – Pattern library for crochet and knitting containing patterns at beginner, intermediate, and advanced levels which also vary from free to paid patterns
* `contact.html` – Contact form for user questions or feedback
* `about.html` – Information about the purpose of the site

## Technologies Used
* HTML5
* CSS3 (`styles.css`)
* JavaScript (`script.js`)

## How to Run
1. Download or clone the project files
2. Open the project folder
3. Open `index.html` in your web browser
